# Weather Forecast Application
A modern, responsive weather application that provides real-time weather information and 5-day forecasts for any location worldwide. Built with React and featuring a sleek glassmorphism design optimized for dark mode.

# Installation

##### 1.Clone the repository

git clone https://github.com/yourusername/weather-forecast-app.git

cd weather-forecast-app

##### 2.Install dependencies

npm install
or
yarn install

##### 3.Create a .env file in the root directory and add your API key

VITE_OPENWEATHER_API_KEY=your_api_key_here

##### 4.Start the development server

npm run dev
or
yarn dev

##### 5.Open your browser and navigate to http://localhost:5173
